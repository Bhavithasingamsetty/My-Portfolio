Real-Time Pressure Mapping System with SRGAN & Raspberry Pi Designed and developed a complete end-to-end system that captures real-time seat pressure data using Raspberry Pi hardware and visualizes it through a dynamic, web-based UI. The system uses a Super-Resolution GAN (SRGAN) to upscale low-resolution sensor input (6√ó6) into a detailed 32√ó32 pressure map. Key Features Hardware-Sensor Integration: Connected a 6√ó6 force sensor matrix to a Raspberry Pi using ADS1115 and TCA9548A to collect live analog pressure data. Real-Time Data Pipeline: Raspberry Pi reads pressure values and transfers them as 6x6_data.csv to the laptop via SSH/SCP every 2 seconds. AI-Based Pressure Upscaling: Trained a PyTorch-based SRGAN model to enhance low-resolution pressure maps to high-resolution (32√ó32) images in real time. Interactive UI for Visualization: Developed a responsive web interface using HTML, CSS, and JavaScript to: ‚Ä¢ Display both the original 6√ó6 and upscaled 32√ó32 pressure maps side by side ‚Ä¢ Refresh images automatically every minute ‚Ä¢ Show timestamp of last update, maximum force, and optionally total pressure Cross-Device Hosting: Flask server hosted locally, allowing access from phones, tablets, and other devices on the same network. Technologies Used Hardware: Raspberry Pi, ADS1115 ADC, TCA9548A Software: Python, PyTorch (SRGAN), Flask, HTML, CSS, JavaScript Data Handling: NumPy, Matplotlib, CSV Communication: SSH, SCP (for real-time Pi ‚Üí laptop transfer)
